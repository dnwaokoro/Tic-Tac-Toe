# Tic Tac Toe

A command-line Tic Tac Toe game built in Java.

## Features
- Two-player turn-based gameplay
- Win-condition checking
- Input validation
- Game-state tracking

## Tech Used
- Java

## How to Run
1. Compile the Java files
2. Run the main file from the command line or IDE

## What I Learned
This project helped me practice control flow, condition checking, and designing interactive programs in Java.
