# Tic Tac Toe

A command-line two-player Tic Tac Toe game built in Java.

## Features
- Two-player turn-based gameplay
- Numpad-style board placement
- Win-condition checking
- Draw tracking and replay prompt

## Tech Used
- Java

## How to Run
1. Compile the file:
   javac Main.java
2. Run the program:
   java Main

## Notes
This project uses console input and game-state logic to manage turns, detect wins, and track results across multiple rounds.

## What I Learned
This project helped me practice control flow, arrays, condition checking, and interactive command-line programming in Java.
